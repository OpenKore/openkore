first				поднять с земли первый попавшийся предмет

[talk]
пошаговый разговор с неписью
<NPC #>				начать разговор с указанной неписью
cont				продолжить слушать непись
resp				вывести список возможных вариантов ответа
resp <response #>	выбрать вариант ответа
num <number>		сказать неписи число
text <string>		сказать неписи текст
no					завершить/отменить разговор с неписью
[talknpc]
поговорить с неписью согласно заданной последовательности
<x> <y> <NPC talk codes>	поговорить со стоящей на <x> <y> неписью, используя <NPC talk codes>
[tank]
танковать мобов для игрока
<player name|player #>		начать танковать мобов для указанного в tankModeTarget игрока
stop				прекратить танковать мобов
[tele]

[timeout]

<type>				вывести значение таймаута <type>
<type> <second>		задать таймауту <type> новое значение <seconds>
[top10]
вывести топ10
top10 (a | alche | alchemist)		вывести список лучших алхимиков
top10 (b | black | blacksmith)		вывести список лучших кузнецов
top10 (p | pk | pvp)				вывести десятку лучших игроков в PVP
top10 (t | tk | taekwon)			вывести десятку лучших тэквондистов
[uneq]

[vender]

<vender #>			войти в указанную торговую лавку
<vender #> <vender item #> [<amount>]	купить указанный предмет в торговой лавке
end				выйти из текущей торговой лавки
